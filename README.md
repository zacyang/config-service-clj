# config-service-clj

A Clojure library designed to provide configuration service in restful way.

## Usage
### use maven 
>lein pom
>import project

### run service
>start repl
>=> (use 'ring.adapter.jetty)
  => (use 'hello-world.core)
  => (run-jetty handler {:port 3000})
